Tech stack used:
Embedded JavaScript, CSS, Node.js, Express.js, MongoDB 

steps to run project locally:
1. download whole code(zip file)
2. extract files from zip file
3. open any editor for example VS code
4. open the file folder
5. Install npm mongoose and express 
6. MongoDB installation is mandatory
7. start your MongoDB in back and run your server
8. adding tasks and deleting tasks reflects in your database
